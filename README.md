# ブロック崩しゲーム 🎮

爆発エフェクトとBGM付きのブロック崩しゲームです。

## 🎯 遊び方

### 操作方法
- **左右矢印キー（←→）**: パドルを左右に移動
- **スペースキー**: ゲームオーバー後にリスタート
- **🔊アイコンクリック**: BGMと効果音のミュート切り替え

### ゲームルール
1. パドルを操作してボールを跳ね返す
2. すべてのブロックを破壊するとクリア
3. ボールを落とすとライフが1つ減る
4. ライフが0になるとゲームオーバー

### 得点
- ブロック破壊: 10点

## 🚀 ゲームを始める

1. `breakout.html`をブラウザで開く
2. ゲーム画面をクリックまたは任意のキーを押してBGMを開始
3. 矢印キーでプレイ開始！

## 🎨 特徴

- **カラフルな爆発エフェクト**: ブロックを破壊すると派手なパーティクルエフェクト
- **ノリノリなBGM**: Web Audio APIで生成された16ビートのBGM
- **効果音**: ブロック破壊、パドルヒット、ミス時の効果音
- **レスポンシブデザイン**: 見やすいスコア表示とゲームオーバー画面

## 🛠️ 技術仕様

- 純粋なHTML/CSS/JavaScript（外部ライブラリ不使用）
- Canvas APIでゲーム描画
- Web Audio APIで音声生成
- パーティクルシステムで爆発エフェクト実装
